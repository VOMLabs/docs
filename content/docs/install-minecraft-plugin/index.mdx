This guide shows you how to install **any** Minecraft server plugin. While we'll use [CraftGuard](https://modrinth.com/plugin/craftguard) as a practical example, these steps work for **every other Minecraft plugin** — as long as it supports the server software you're running (Paper, Purpur, Folia, Spigot, etc.).

## What is a Minecraft Plugin?

Plugins extend your server's functionality by adding new features, commands, and gameplay modifications. They run on supported server platforms like Paper, Purpur, or Spigot.

## Requirements

Before installing plugins, ensure your server meets these requirements:

- **Server Platform**: Paper, Purpur, Folia, or another compatible server JAR
- **Java**: Version 17+ (some plugins require Java 21+)

<Callout type="info">
  Most modern plugins require Paper or its forks (Purpur, Folia) for best
  compatibility.
</Callout>
